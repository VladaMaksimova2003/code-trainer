Toxic Pulse — макет платформы

Единая тёмная дизайн-система для платформы обучения программированию. Лайм #8eff01 — основной акцент, фиолетовый #8b53fe — вторичный, поверхности на базе #1d2331. React + Vite + Tailwind. Этот файл — визуальный референс для Cursor AI; токены и карта экранов в сопроводительной спецификации.

Двоичный поиск

Дан отсортированный массив целых чисел arr и число x. Верните индекс x или −1, если элемента нет. Сложность — O(log n).

Пример 1
›Вход:  arr=[1,3,5,7,9], x=5
›Выход: 2
Ограничения
  • 1 ≤ n ≤ 10⁵
  • массив отсортирован по возрастанию
  • −10⁹ ≤ arr[i] ≤ 10⁹
Код (Monaco)БлокиБлок-схема
1def binary_search(arr, x):
2    lo, hi = 0, len(arr) - 1
3    while lo <= hi:
4        mid = (lo + hi) // 2
5        if arr[mid] == x: return mid
6        elif arr[mid] < x: lo = mid + 1
7        else: hi = mid - 1
8    return -1
