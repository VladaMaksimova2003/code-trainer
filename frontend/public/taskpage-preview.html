Code Trainer Алгоритм medium адаптив
Напишите функцию sum_digits(n), которая принимает целое число n и возвращает сумму его цифр. Например: sum_digits(123) → 6 sum_digits(0) → 0 sum_digits(99) → 18
Как решить
Преобразуй число в строку, пройдись по каждому символу и суммируй int() от каждой цифры.
Конструкции
Function Loop Return 
def sum_digits(n):
    # преобразуем в строку
    return sum(int(d) for d in str(abs(n)))
